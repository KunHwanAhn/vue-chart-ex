# vue-chart-ex

## 프로젝트 설명
vue.js에서 사용할 Chart Library로 Chart.js를 선택한 뒤, Chart.js의 Vue.js용 Wrapper인 [vue-chartjs](https://vue-chartjs.org/)를 사용하는 것이 좋을지, [Chart.js](https://www.chartjs.org/docs/latest/) 자체를 사용하는 것이 좋을지 비교하기 위하여 만든 프로젝트이다.
